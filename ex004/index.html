Enunciado

Faça um script que receba uma data no formato “dd/mm/aaaa” e escreva a data por extenso.

Dica: use a função “split” de uma string que quebra a string em pedaços dado um separador como argumento da função. Nesse caso, o separador é a barra (/) da data. Exemplo: Para a entrada “22/04/2019” deve ser escrito “22 de abril de 2019”.

(Obs.: não necessita de laço de repetição).
